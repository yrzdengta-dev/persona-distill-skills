# Examples

## Example 1

User input:

```text
Use daancrypto-perspective to judge whether BTC is at a range edge or stuck in the middle.
```

Expected output traits:

- identify the range first
- list key levels
- prefer waiting if price is mid-range

## Example 2

User input:

```text
Analyze this ETH breakout with daancrypto-perspective and focus on retest confirmation plus invalidation.
```

Expected output traits:

- breakout is not accepted automatically
- define the retest condition
- include a precise invalidation boundary
