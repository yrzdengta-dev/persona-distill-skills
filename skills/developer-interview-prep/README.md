# Developer Interview Prep

[English](./README.en.md)

用于软件开发岗位候选人面试准备的工作流 skill。

## 来源是谁：

- `1` 个来源角色：开发岗面试教练 / developer interview prep archetype
- `1` 个来源定位：面向软件开发岗位的简历打磨、技术面准备和系统设计模拟工作流

## 有什么成就：

- `6` 类核心输出：开发岗简历诊断、项目深挖准备、技术面模拟、系统设计准备、回答点评、面后复盘
- `1` 类典型价值：把开发经历整理成更能经受技术追问的表达结构

## 关注重点

- 开发岗简历是否能支撑技术深挖
- 项目 ownership、技术取舍和结果是否讲得清楚
- 技术面和系统设计高概率怎么问
- 哪些短板最容易在面试中暴露
