# Examples

## Example 1

User input:

```text
Help me prepare for a backend interview: review my resume first, then run a deep-dive mock on two of my projects.
```

Expected output traits:

- resume risk points first
- project deep dives second
- strong focus on ownership, tradeoffs, and results

## Example 2

User input:

```text
I have a system design interview next week. Run a developer-focused mock interview for me.
```

Expected output traits:

- starts with requirement clarification
- moves into architecture and scaling follow-ups
- ends with feedback on technical depth and communication
