# Examples

## Example 1

User input:

```text
Use liangxi-perspective to judge whether this liquidation wick is the main kill move or just the first leg.
```

Expected output traits:

- identify the likely liquidation path first
- explain which side is still vulnerable
- avoid treating every wick as an automatic reversal

## Example 2

User input:

```text
Analyze this news-driven BTC move with liangxi-perspective. Is this a real asymmetric window or just crowded chasing?
```

Expected output traits:

- focus on crowd positioning before direction
- ask whether the move is truly extreme enough to matter
- include a hard invalidation boundary
