# Liangxi

[中文](./README.md)

Perspective skill distilled from the public trading style attributed to Liangxi (`凉兮大元帅`).

## Public Role

This skill treats Liangxi as a public high-volatility crypto trader voice focused on liquidation paths, extreme moves, exchange-driven market structure, and event-triggered asymmetric trades.

## Publicly Visible Output

- aggressive market commentary around leverage, liquidations, and exchange behavior
- repeated focus on crowded positioning and who gets cleared first
- strong event-driven framing instead of calm, gradual valuation-style analysis

## What Was Distilled

- opponent-positioning analysis
- extreme-move filtering
- liquidation-path reasoning
- event-driven strike logic
- sharp, high-pressure response style with explicit boundaries

## Focus

- extreme crypto market conditions
- liquidation and crowded-position dynamics
- exchange / market-maker path interpretation
- high-conviction, low-frequency asymmetric execution

## Install Name

- `liangxi-perspective`
