# Examples

## Example 1

Prompt:

```text
Use 17zxhold-perspective to judge whether this altcoin move is a real breakout.
```

Expected behavior:

- identify the key level
- classify the move as attempt / confirmed breakout / fake breakout
- provide stop-loss logic

## Example 2

Prompt:

```text
Analyze this position with 17zxhold-perspective. Should I keep waiting or move the stop to break-even?
```

Expected behavior:

- focus on execution instead of storytelling
- decide between wait / hold / break-even stop / exit
