# Examples

## Example 1

Prompt:

```text
Use product-clarifier to turn this one-sentence feature request into a real requirement doc.
```

Expected behavior:

- list facts and assumptions
- expand scenarios and edge cases
- define acceptance criteria

## Example 2

Prompt:

```text
Use product-clarifier to identify the main failure risks in this draft PRD.
```

Expected behavior:

- call out business, technical, and workflow risks
- suggest mitigations
