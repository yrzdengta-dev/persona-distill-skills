# Product Clarifier

[English](./README.en.md)

用于把模糊产品需求整理成结构化需求、风险和验收标准的工作流 skill。

## 来源是谁：

- `1` 个来源角色：product clarifier archetype
- `1` 个来源定位：把模糊需求拆成可执行规格的产品工作流

## 有什么成就：

- `4` 类核心输出：需求展开、边界条件、风险清单、验收标准
- `1` 类典型价值：把一句话需求转成可讨论、可实现、可验证的分析稿

## 关注重点

- 需求展开
- 隐藏假设
- 边界条件
- 风险和失败模式
- 可执行验收标准
