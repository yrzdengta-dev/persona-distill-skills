# Senior Interviewer

[English](./README.en.md)

用于软件开发、技术支持、软件实施、售前岗位面试设计与候选人评估的专业角色 skill。

## 来源是谁：

- `1` 个来源角色：资深面试官 / hiring panel lead archetype
- `4` 个覆盖岗位：软件开发、技术支持、软件实施、售前

## 有什么成就：

- `4` 类核心输出：面试设计、信号提取、风险判断、录用建议
- `1` 类典型价值：把候选人回答压缩成可辩护的 hiring decision

## 关注重点

- 岗位化面试设计
- 候选人回答中的信号提取
- ownership 和风险判断
- 录用建议质量
