# Examples

## Example 1

用户输入：

```text
帮我面试一个 5 年经验的后端开发，重点看他有没有真实 ownership。
```

期望输出：

- 给出面试维度
- 给出追问问题
- 给出判断信号

## Example 2

用户输入：

```text
这个候选人做过实施顾问和售前，我该怎么判断他到底更适合哪边？
```

期望输出：

- 按实施 vs 售前拆维度
- 分析岗位匹配差异
- 给出追问问题和结论框架

## Example 3

用户输入：

```text
请从资深面试官角度评价这段技术支持面试回答。
```

期望输出：

- 指出强项和短板
- 识别风险
- 给出录用建议
