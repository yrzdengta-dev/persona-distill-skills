# ICT

[English](./README.en.md)

从公开 ICT / Smart Money Concepts 框架中提炼的方法论 skill。

## 人物介绍

这个 skill 把 ICT 视为公开交易教育者和方法论来源，而不是人格扮演对象。

## 公开成果与可见输出

当前版本主要基于 `3` 类公开来源、至少 `5` 个高频教学主题进行蒸馏：

- 公开 ICT / SMC 教学材料
- 高频讲授流动性、PD Array 和 session model
- 在 discretionary intraday 社群中被广泛引用的方法体系

## 蒸馏过程中采集了什么

当前版本显式采集了 `5` 类内容：

- 公开方法论定位
- 高频概念词汇与核心模型
- 会话与时间维度规则
- 执行顺序和分析优先级
- 社群中被反复引用的核心分析框架

## 提炼出了什么

当前版本提炼出 `5` 个核心模块：

- HTF premise 顺序
- 流动性目标优先分析
- PD Array 上下文
- AMD / Judas / displacement / rebalancing
- 结合 session 的执行逻辑

## 关注重点

- HTF premise
- liquidity targets
- PD Arrays
- AMD / Judas swing / displacement / rebalancing
- session-aware execution
