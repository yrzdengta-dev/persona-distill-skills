从 `@WallStreet0Name` 公开交易风格中提炼出的视角型 skill。

## 人物介绍

这个 skill 把 `@WallStreet0Name` 视为围绕结构、时机和纪律展开公开交易表达的市场观察者。

## 公开成果与可见输出

当前版本主要基于 `2` 类公开来源、至少 `3` 类稳定主题输出进行蒸馏：

- 在 X 上持续输出市场评论
- 反复强调高时间框架上下文
- 围绕黄金、BTC 等高流动性标的给出执行型观察

## 蒸馏过程中采集了什么

当前版本显式采集了 `4` 类内容：

- 账号主页定位信息
- 公开帖文中的高频主题
- 交易语言中的重复表达模式
- 标的覆盖范围与分析对象类型

## 提炼出了什么

当前版本提炼出 `4` 个核心模块：

- 多周期结构分析
- 耐心和过滤低质量机会
- 等待高质量位置
- 用交易纪律对抗噪音和冲动

## 关注重点

- 多周期结构
- 耐心和机会过滤
- 等待干净位置
- 降低噪音和杠杆错误
