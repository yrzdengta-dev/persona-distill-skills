# Interview Prep Coach

[English](./README.en.md)

用于候选人面试准备的工作流 skill。

## 来源是谁：

- `1` 个来源角色：候选人面试教练 / interview prep coach archetype
- `1` 个来源定位：面向求职准备、材料打磨和模拟面试的角色化工作流

## 有什么成就：

- `5` 类核心输出：简历诊断、岗位匹配分析、面试准备提纲、模拟面试、面后复盘
- `1` 类典型价值：把零散经历整理成更适合求职表达与面试回答的结构

## 关注重点

- 简历是否清楚、真实、匹配岗位
- 项目经历和行为面回答如何讲得更扎实
- 高概率问题和追问该怎么准备
- 模拟面试后该优先补哪些短板
