# Examples

## Example 1

User input:

```text
Help me prepare for a backend interview: review my resume first, then run a mock technical interview.
```

Expected output traits:

- resume review comes first
- likely questions are tailored to the role
- mock round includes follow-up pressure and feedback

## Example 2

User input:

```text
I just finished an implementation consultant interview and my answers felt weak. Help me debrief and rewrite my responses.
```

Expected output traits:

- identify why the answers felt weak
- rewrite them into clearer structures
- end with specific next-step preparation advice
