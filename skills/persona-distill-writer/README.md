# Persona Distill Writer

Workflow skill for turning fragmented materials into a stable persona, style profile, or framework description.

## Focus

- extracting voice and style patterns
- separating facts from inference
- summarizing recurring decision patterns
- producing reusable persona documents

## Install Name

- `persona-distill-writer`

## Included Files

- `SKILL.md`
- `examples.md` / `examples.en.md`
